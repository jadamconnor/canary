About This App

What is this?
This is Canary, an app that uses a mathematical formula known as phi coefficient to determine whether specific symptoms are correlated to specific events.
How should I use Canary?
Each day you should input the day's events and symptoms in the corresponding fields. You should use short terms for each. There's no limit on length of terms, but you must always use the same term for each. Canary's logic correlates terms, so short, memorable terms are best. Don't worry, Canary features autocomplete, which will help you keep terms consistent.
Then what?
Once Canary has collected enough data, data will be displayed in the Data View section. If a given event is found to correlate with a given symptom it will be displayed below the symptom. Events shown in red are positively correlated, meaning they occur together. Events shown in green are negatively correlated, meaning they tend to be mutually exclusive.
Anything else?
It should be said that correlation does not demonstrate causality. That certain events correlate with certain symptoms does not prove the former causes the latter. However, correlation is an important step in identifying causality. In that way, Canary can be a helpful tool.
